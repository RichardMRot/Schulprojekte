How to use this?

You put the (text with) chords that you want transposed in the upper box. We will try to automatically find the scale that it is in, but if it is wrong you can set it to the correct scale with the box in the top-right.

Next you can either set by how many steps the chords should be transposed or into which scale it should be transposed.

Lastly, click the "Transpose" button in the bottom and it will be transposed!

If your input contains text, make sure to have the text-lines be separated from the chord-lines, otherwise the result may yield wrong results!
