How to use this?

You put the (text with) chords that you want transposed in the upper box. Next you can set by how many steps the chords should be transposed. Lastly, click the "Transpose" button in the bottom and it will be transposed!

If your input contains text, make sure to have the text-lines be separated from the chord-lines, otherwise the result may yield wrong results!

Also, if the result contains, for example flats (b), but you want sharps (#), just use the 'Enharmonic Change' button. It also says to which types of notes it is currently toggled to in the parantheses.
